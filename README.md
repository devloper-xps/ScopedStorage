# ScopedStorage
Scoped Storage Demo in Android
<br>
### Includes:
Download Image.<br>
Media Store example.<br>
Delete Image with RecoverableSecurityException.<br>
<ACCESS_MEDIA_LOCATION> permission example.<br>
OPEN_DOCUMENT_TREE example.<br>
